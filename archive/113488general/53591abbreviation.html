Sebastien Gouezel (Dec 28 2019 at 18:57):

I have tried to convert module to an abbreviation for semimodule, but something is not working out of the box. Here is a minimal example showing the problem.

import algebra.pi_instances

universes u v w

class SemiModule (α : Type u) [semiring α] (β : Type v) [add_comm_monoid β]

abbreviation Module (α : Type u) [ring α] (β : Type v) [add_comm_group β] :=
SemiModule α β

instance piSemiModule (I : Type w) (f : I → Type u) (α : Type v)
  [semiring α]  [∀ i, add_comm_monoid $ f i] [∀ i, SemiModule α $ f i] :
  SemiModule α (Π i : I, f i) := by constructor

instance piModule (I : Type w) (f : I → Type u) (α : Type v)
  [ring α] [∀ i, add_comm_group $ f i] [∀ i, Module α $ f i] :
  Module α (Π i : I, f i) := by apply_instance

The last by apply_instance does not find the instance piSemiModule which is right here. I don't understand why.
